FAQ

You've run into an issue with filterbubble.ga or have something to ask? Read our FAQ below

General 

1. Haven't used Twitter before?

If you are not currently a Twitter user, you can start by searching for the Twitter account information of the name of the person or organisation you are interested in and then start using the features of this website.

2. How can I be sure that the Twitter account name I enter is the one I am looking for?

As shown in the picture, Twitter Handle follows the “@” symbol in the profile. You can search the homepage of the person or group that you are interested in to check if you input their unique Tweet handle name.

3. Which language is EYE OPENER in?

The interface of our program is only available in English at the moment. No other languages are supported for the interface.

4. If I search my own Twitter account using this site's function, will my information be collected?

filterbubble.ga does not collect personal information from you, so please feel free to use it. 

5. I have found a bug. What do I do?

Report it and we will fix it! We got EYE OPENER from zero to where it is now, and we won't stop improving it.

Common Issues

1. Why does it take so long to load the data for analysis?

Please understand that we are currently a student team and therefore have limited resources and hardware at our disposal. We need the support of investors to make further improvements.
